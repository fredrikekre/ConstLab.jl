CA2: Prototype model for plasticity¶

Fredrik Ekre, fredrik.ekre@chalmers.se

In this assignment we implement a prototype model of plasticity; Hooke elasticity and von Mises yield surface with non-linear mixed saturation hardening. The model follows Section 6.3 of Constitutive Modeling of Engineering Materials[1]. The implementation is available in the ConstLab GitHub repository and this notebook only presents some usage and results.

[1] Constitutive Modeling of Engineering Materials - Theory and Computation, Runesson, Steinmann, Ekh, Menzel

Example 1¶

In the first example we replicate the results given for cyclic loading in Figure 6.5 in [1]. The material parameters are: $E = 210\ \mathrm{GPa}$, $\nu = 0.3$, $H = 6.5\ \mathrm{GPa}$, $\sigma_\mathrm{Y} = 543\ \mathrm{MPa}$, $\alpha_\infty = 132\ \mathrm{MPa}$, $\kappa_\infty = 228\ \mathrm{MPa}$. The $r$ parameter is set to $r = 0$ (pure kinematic hardening), $r = 0.5$ (mixed hardening) and $r = 1$ (pure isotropic hardening). The specimen is under uniaxial stress ($\sigma_{22} = \sigma_{33} = \sigma_{12} = \sigma_{13} = \sigma_{23} = 0$) and loaded with a cyclic prescribed strain $\varepsilon_{11} = \varepsilon_{\mathrm{amp}} \sin(t)$ with $\varepsilon_{\mathrm{amp}} = 0.02$.

In the second example we apply a cyclic stress: $\sigma_{11}(t) = \sigma_0 + \sigma_{\mathrm{amp}} \sin(t)$ while keeping the remaining stress components to $0$ and aim to simulate ratchetting. The material response is plotted, and also the accumulated plastic strain.

Equivalent plastic strain¶

The equivalent plastic strain is defined as

$$ \bar{\varepsilon}^p(\tau) = \int_0^\tau \dot{\bar{\varepsilon}}^p\ \mathrm{d}t = \int_0^\tau \sqrt{\frac{3}{2}}|\dot{\varepsilon}^p|\ \mathrm{d}t. $$

For the discrete timestepping scheme this can be approximated as as sum of one term per time-interval:

$$ \bar{\varepsilon}^p \approx \sum_{i=2}^N\ \sqrt{\frac{3}{2}}|\frac{\dot{\varepsilon}^p_i - \dot{\varepsilon}^p_{i-1}}{\Delta t}|\ \Delta t = \sum_{i=2}^N\ \sqrt{\frac{3}{2}}|\dot{\varepsilon}^p_i - \dot{\varepsilon}^p_{i-1}| $$

acc = [Float64[0.0] for _ in 1:3]
for (i, res) in enumerate((res1, res2, res3))
    itr = (r.εₚ for r in res)
    for (ε, εₙ) in zip(Iterators.drop(itr, 1), itr)
        r = √(3/2) * norm(ε - εₙ) # * Δt / Δt
        push!(acc[i], r + acc[i][end])
    end
end
